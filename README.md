# Executive Operations Dashboard

An AI-powered operations dashboard designed for service companies to monitor KPIs, automate reporting, and generate executive insights.

## Features

- Executive KPI Dashboard
- Operational Analytics
- Automated Reporting
- AI Generated Summaries
- PostgreSQL Database
- FastAPI Backend

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- Power BI
- Docker
